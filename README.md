# Real-Time Eye-Tracking for Cursor Control Using OpenCV

A computer vision project that enables hands-free cursor control using eye movements and blink gestures.

## Features

* Real-time eye tracking
* Cursor movement using gaze direction
* Single Blink → Right Click
* Double Blink → Scroll Down
* Long Blink → Scroll Up
* Dwell Gaze → Left Click
* Adaptive EAR Calibration

## Technologies Used

* Python 3.10
* OpenCV
* MediaPipe
* NumPy
* PyAutoGUI

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
